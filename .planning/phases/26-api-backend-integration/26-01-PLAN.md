```python
class SearchScope(str, Enum):
    """6 scope levels for contextual search."""
    GLOBAL = "global"           # All institutions
    INSTITUTION = "institution" # Single institution, all content
    PROGRAM = "program"         # Single program within institution
    DOCUMENT = "document"       # Single document context
    STANDARDS = "standards"     # Standards library (accreditor-scoped)
    COMPLIANCE = "compliance"   # Compliance findings/audits

@dataclass
class SearchContext:
    """Captures user's current location for search scoping."""
    scope: SearchScope
    institution_id: Optional[str] = None
    program_id: Optional[str] = None
    document_id: Optional[str] = None
    accreditor_id: Optional[str] = None

    def to_dict(self) -> Dict[str, Any]:
        return {
            "scope": self.scope.value,
            "institution_id": self.institution_id,
            "program_id": self.program_id,
            "document_id": self.document_id,
            "accreditor_id": self.accreditor_id,
        }
```

**POST /api/search/contextual** (per D-01, D-02, D-05):
- Accept JSON body: `{query, scope, institution_id?, program_id?, document_id?, accreditor_code?, sources[]?, page?, per_page?, semantic?}`
- Validate: `query` required (return 400 if missing)
- Validate scope requirements (per D-05):
  - `INSTITUTION` requires `institution_id` (400 if missing)
  - `PROGRAM` requires `institution_id` and `program_id`
  - `DOCUMENT` requires `institution_id` and `document_id`
  - `STANDARDS` optionally uses `accreditor_code` (map to accreditor_id via standards_store)
  - `COMPLIANCE` requires `institution_id`
- Create SearchContext from request body
- Call `get_contextual_search_service(context, _workspace_manager).search(query, sources, limit, offset)`
- Build facets by aggregating results.source_type counts
- Return JSON (per D-02):
```json
{
  "query": "...",
  "scope": "institution",
  "total": 42,
  "items": [...],
  "facets": {"documents": 10, "standards": 5, ...},
  "page": 1,
  "per_page": 20,
  "context": {"scope": "institution", "institution_id": "...", ...}
}
```
- Default pagination: page=1, per_page=20 (per Claude's Discretion)

**GET /api/search/contextual/sources** (per D-03):
- Accept query param: `scope` (required)
- Validate scope is valid SearchScope value
- Return available sources for that scope:
  - GLOBAL: all 8 sources
  - INSTITUTION/PROGRAM/DOCUMENT/COMPLIANCE: all 8 sources
  - STANDARDS: ["standards"] only
- Return JSON: `{"scope": "...", "sources": [...]}`

**GET /api/search/contextual/suggest** (per D-04):
- Accept query params: `scope`, `institution_id?`, `prefix?`
- Return recent searches and common queries for scope
- Query `site_visit_searches` table filtered by scope
- Return JSON: `{"suggestions": [{"query": "...", "count": N}, ...]}`
