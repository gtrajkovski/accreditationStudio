  <action>
Modify `templates/base.html` to add data attributes to the main container (per D-07).

Find the `<div class="main-wrapper">` element (approximately line 295) and add data attributes:

```html
<div class="main-wrapper"
     data-page="{% block data_page %}{{ request.endpoint|replace('_', '-') if request.endpoint else 'unknown' }}{% endblock %}"
     data-institution-id="{% block data_institution_id %}{{ current_institution.id if current_institution else '' }}{% endblock %}"
     data-program-id="{% block data_program_id %}{% endblock %}"
     data-document-id="{% block data_document_id %}{% endblock %}"
     data-accreditor="{% block data_accreditor %}{% endblock %}">
```

This uses Jinja2 blocks so child templates can override specific values.

Default behavior:
- `data-page`: Uses Flask endpoint name with underscores replaced by hyphens
- `data-institution-id`: Uses current_institution.id if available (set by context processor)
- `data-program-id`: Empty by default, set by program pages
- `data-document-id`: Empty by default, set by document pages
- `data-accreditor`: Empty by default, set by standards pages
