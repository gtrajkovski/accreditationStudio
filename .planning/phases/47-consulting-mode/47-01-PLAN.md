# Plan 47-01: Consulting Mode & Readiness Assessment

## Objective

Implement consulting replacement workflows: readiness assessment, pre-visit checklist, and guided self-assessment.

## Strategic Context

Consulting Mode replaces the work of $150-300/hr accreditation consultants. Key selling point for career schools currently paying $15K-40K/year for consultant services.

## Dependencies

- Phase 42 (RBAC) for permissions

## Tasks

### Task 1: Consulting Service

**File:** `src/services/consulting_service.py`

**1. Readiness Assessment Generator:**
`generate_readiness_assessment(institution_id, accreditor_code)` → structured report

Pulls from: readiness score, audit findings, document inventory, task status, faculty credentials, student outcomes

Produces:
- Overall rating: Ready / Conditionally Ready / Not Ready
- Section-by-section compliance status
- Critical gaps requiring attention
- Timeline recommendation
- Estimated remediation effort

**2. Pre-Visit Checklist Generator:**
`generate_pre_visit_checklist(institution_id, accreditor_code)` → checklist items

Auto-populated from audit findings, document status, evidence coverage.

Organized by ACCSC evaluation areas:
- Administration & Management
- Academics
- Admissions
- Student Services
- Financial Stability
- Facilities & Equipment
- Catalog & Publications
- Student Achievement

Each item: requirement, status (met/not met/partial), evidence reference, action needed

**3. Guided Self-Assessment:**
`get_self_assessment_questions(accreditor_code, section=None)` → questions with guidance

For each requirement:
- Standard text
- What accreditor looks for
- Evidence to prepare
- Common deficiencies
- AccreditAI's current assessment

**Commit:** `feat(47): implement consulting service`

### Task 2: Consulting API Blueprint

**File:** `src/api/consulting.py`

Blueprint: `consulting_bp`, prefix `/api/consulting`

Endpoints:
- `GET /readiness-assessment/<institution_id>` — generate (compliance_officer+)
- `POST /readiness-assessment/<institution_id>/export` — PDF (compliance_officer+)
- `GET /pre-visit-checklist/<institution_id>` — generate (compliance_officer+)
- `POST /pre-visit-checklist/<institution_id>/export` — DOCX (compliance_officer+)
- `GET /self-assessment/<institution_id>` — sections (department_head+)
- `GET /self-assessment/<institution_id>/<section>` — section detail (department_head+)
- `POST /self-assessment/<institution_id>/<section>/complete` — mark reviewed (compliance_officer+)

**Commit:** `feat(47): add consulting API blueprint`
