### Task 2: Update AIClient with Model Override

**File:** `src/ai/client.py`

**Changes:**
1. Add `model` parameter to `chat()`, `chat_stream()`, `generate()` methods
2. Default to `self.model` if not specified
3. Add helper methods `generate_fast()` and `generate_reasoning()`

**Implementation:**
```python
def generate(
    self,
    system_prompt: str,
    user_prompt: str,
    max_tokens: Optional[int] = None,
    model: Optional[str] = None  # NEW: model override
) -> str:
    """Generate a one-shot response without polluting conversation history."""
    response = self.client.messages.create(
        model=model or self.model,  # Use override or default
        max_tokens=max_tokens or self.max_tokens,
        system=system_prompt,
        messages=[{"role": "user", "content": user_prompt}]
    )
    return response.content[0].text

def generate_fast(self, system_prompt: str, user_prompt: str) -> str:
    """Generate using fast model (Haiku) for simple tasks."""
    return self.generate(
        system_prompt=system_prompt,
        user_prompt=user_prompt,
        model=Config.MODEL_FAST,
        max_tokens=Config.MAX_TOKENS_FAST
    )
```

**Verification:**
- `generate_fast()` uses Haiku model
- Default `generate()` still uses Sonnet
