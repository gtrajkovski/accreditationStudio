  <name>Task 1: EvidenceAssistantService with context-aware search</name>
  <files>src/services/evidence_assistant_service.py, tests/test_evidence_assistant_service.py</files>
  <action>
1. Create `src/services/evidence_assistant_service.py`:
   - Class `EvidenceAssistantService` with `__init__(search_service, explainer_service, ai_client)`

   - Method `find_evidence_for_standard(institution_id: str, standard_id: str, query: str = None, context: dict = None) -> List[EvidenceResult]`:
     - Get required_evidence types from explainer_service.explain_standard(standard_id)
     - Build search query: use provided query OR first 500 chars of standard body
     - Call search_service.search() with n_results=20
     - Apply weighting: boost score by 1.5x for documents matching required_doc_types
     - Return top 10 as EvidenceResult objects with fields:
       - document_id, doc_type, snippet, page, score, is_required_type, relevance_label
     - relevance_label: "Required" (is_required_type=True), "Relevant" (score > 0.7), "Related" (else)

   - Method `generate_suggested_prompts(conversation_history: List[Dict], context: Dict) -> List[str]`:
     - Build context string from: current_page, active_standard_id, recent_findings
     - Call ai_client.generate() with system prompt to produce 3-5 specific follow-up questions
     - Parse JSON response, return list of prompt strings
     - Handle parse errors gracefully (return empty list)

   - Dataclass `EvidenceResult`:
     - document_id: str, doc_type: str, snippet: str, page: Optional[int]
     - score: float, is_required_type: bool, relevance_label: str
     - confidence: float (same as score, normalized 0-1)
     - to_dict() method

2. Create `tests/test_evidence_assistant_service.py`:
   - Mock search_service.search() to return sample SearchResults
   - Mock explainer_service.explain_standard() to return required_evidence list
   - Test find_evidence_for_standard:
     - Returns EvidenceResult objects with required fields
     - Documents with required types have higher scores (boosted)
     - Results sorted by score descending
   - Test generate_suggested_prompts:
     - Returns list of strings
     - Mock AI response, verify parsing
     - Test graceful failure on invalid JSON
