# Plan 45-01: Executive Dashboard

## Objective

Create executive-level dashboard with aggregated metrics, readiness trends, AI attention summary, and upcoming deadlines.

## Dependencies

- Phase 42 (RBAC) for admin-only access
- Phase 44 (Tasks) for task metrics

## Tasks

### Task 1: Readiness Snapshots Table

**Add to migration or create `src/db/migrations/0050_readiness_snapshots.sql`:**

```sql
CREATE TABLE IF NOT EXISTS readiness_snapshots (
    id TEXT PRIMARY KEY,
    institution_id TEXT NOT NULL,
    score REAL NOT NULL,
    documents_score REAL,
    compliance_score REAL,
    evidence_score REAL,
    consistency_score REAL,
    recorded_at TEXT NOT NULL DEFAULT (datetime('now')),
    FOREIGN KEY (institution_id) REFERENCES institutions(id)
);

CREATE INDEX IF NOT EXISTS idx_readiness_snapshots_inst
    ON readiness_snapshots(institution_id, recorded_at);
```

**Commit:** `chore(45): add readiness_snapshots table`

### Task 3: Executive API Blueprint

**File:** `src/api/executive.py`

Blueprint: `executive_bp`, prefix `/api/executive`

Endpoints:
- `GET /overview` — aggregated metrics (admin+)
- `GET /trends` — readiness trend (admin+)
- `GET /attention` — AI-generated summary (admin+)

**Overview response:**
```json
{
  "readiness": {"current": 72, "previous": 65, "trend": "up"},
  "documents": {"total": 45, "audited": 38, "compliant": 30, "needing_attention": 8},
  "findings": {"total": 127, "resolved": 98, "open_critical": 3, "open_major": 12, "open_minor": 14},
  "tasks": {"total": 42, "completed": 28, "overdue": 5, "due_this_week": 8},
  "upcoming_deadlines": [{"title": "ACCSC Annual Report", "due": "2026-05-15", "days_remaining": 47}],
  "recent_activity_count": 156,
  "last_audit_date": "2026-03-25"
}
```

**Attention endpoint:** Use AI client to generate 3-5 sentence natural language summary. If AI unavailable, use template-based summary.

Example: "Your institution's readiness score has improved 7 points to 72% this month. However, 3 critical findings remain unresolved in your enrollment agreement compliance. 5 tasks are overdue — prioritize the EA audit findings before the April submission deadline."

**Commit:** `feat(45): add executive API blueprint`
