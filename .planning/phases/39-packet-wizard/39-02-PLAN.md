# Plan 39-02: Packet Studio Wizard UI

## Objective

Create 5-step visual wizard UI with step navigation, progress indicators, and live preview.

## Template

File: `templates/packet_wizard.html`

5-step wizard with:
1. Submission type cards (self-study, response, teach-out, annual, substantive change)
2. Standards tree with checkboxes and section shortcuts
3. Evidence mapping with drag-drop and AI suggest
4. Narrative editor with generate/edit/regenerate
5. Live preview with export button

## JavaScript

File: `static/js/packet_wizard.js`

- Step state machine
- Form validation per step
- Drag-drop evidence mapping
- SSE for narrative generation
- Live preview rendering

## CSS

File: `static/css/packet_wizard.css`

- Wizard step indicators
- Card selection styles
- Tree view for standards
- Drag-drop zones
- Preview panel

## Acceptance Criteria

- [ ] 5-step wizard navigable via back/next
- [ ] Step indicators show progress
- [ ] Submission type cards selectable
- [ ] Standards tree with checkboxes
- [ ] Evidence drag-drop to standards
- [ ] AI suggest evidence works
- [ ] Narrative editor with generate button
- [ ] Live preview updates
- [ ] Export button appears on step 5
- [ ] All labels translated
