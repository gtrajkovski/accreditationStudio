       - Call ReportService.create_template(...)

    2. GET /api/reports/templates?institution_id=X
       - Query param: institution_id (required)
       - Call ReportService.list_templates(institution_id)
       - Return 200 with {success: true, templates: [...], count}
       - Error handling: 400 if missing institution_id, 500 for exceptions

    3. GET /api/reports/templates/:template_id
       - Call ReportService.get_template(template_id)
       - Return 200 with {success: true, template: {...}}
       - Return 404 if template not found
       - Error handling: 500 for exceptions

    4. PATCH /api/reports/templates/:template_id
       - Body: {name?, sections?, description?, is_default?}

       - Call ReportService.update_template(template_id, **fields)
       - Return 200 with {success: true, message: "Template updated"}
       - Return 404 if template not found
       - Error handling: 400 for validation, 500 for exceptions

    5. DELETE /api/reports/templates/:template_id
       - Call ReportService.delete_template(template_id)
       - Return 200 with {success: true, message: "Template deleted"}
       - Return 404 if template not found
       - Error handling: 500 for exceptions
