## API Endpoints

- GET /api/documents/{id}/ide-view → document with finding positions
- GET /api/documents/{id}/findings/positions → finding positions only
- GET /api/findings/{id}/preview-fix → remediation preview
- POST /api/findings/{id}/apply-fix → apply fix
- GET /api/documents/{id}/diff → original vs remediated

## Acceptance Criteria

- [ ] Finding positions calculated from text
- [ ] Remediation preview available before applying
- [ ] Apply fix creates remediated version
- [ ] Diff shows changes inline
