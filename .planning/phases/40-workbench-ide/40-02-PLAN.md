# Plan 40-02: Document Workbench IDE UI

## Objective

Create three-panel IDE layout with document list, content viewer with inline markers, and findings panel.

## Template

File: `templates/workbench_ide.html`

Three-panel layout:
- Left (20%): Document list with severity badges
- Center (50%): Document viewer with page nav and inline highlights
- Right (30%): Findings panel with details and actions

## JavaScript

File: `static/js/workbench_ide.js`

- Panel resize
- Finding click → scroll to position
- Inline highlight rendering
- Fix button → remediation
- View mode toggle (original/remediated/diff)

## CSS

File: `static/css/workbench_ide.css`

- Three-panel flex layout
- Resizable panels
- Inline highlight styles
- Finding marker styles
- Diff view styles

## Acceptance Criteria

- [ ] Three-panel layout renders correctly
- [ ] Document list shows severity badges
- [ ] Clicking finding scrolls to position
- [ ] Inline highlights visible
- [ ] Fix button triggers remediation
- [ ] View mode toggle works
- [ ] Diff mode shows changes
