# Plan 20-03: Autopilot UI

## Goal
Add autopilot controls and morning brief display to the Command Center dashboard.

## Tasks

### 1. Last Autopilot Run Card
- **File:** `templates/dashboard.html`
- **Location:** Command Center section
- **Content:**
  - Last run timestamp + status
  - Docs indexed / failed
  - Readiness delta
  - "Run Now" button

### 2. Morning Brief Panel
- **File:** `templates/dashboard.html` or `templates/partials/morning_brief.html`
- **Content:**
  - Readiness score ring with delta indicator
  - Top 3 blockers (expandable)
  - Next 3 actions (with links)
  - "View Full Brief" link
  - Download button

### 3. Autopilot Settings
- **File:** `templates/institution_settings.html` (or section in institution detail)
- **Fields:**
  - Enable/disable toggle
  - Schedule time (hour:minute)
  - Task toggles: reindex, consistency, audit, readiness
  - Notification preferences

### 4. JavaScript Controller
- **File:** `static/js/autopilot.js`
- **Functions:**
  - `runNow()` — POST to run-now, connect SSE, show progress modal
  - `loadLatestBrief()` — GET brief, render panel
  - `saveSettings()` — PUT settings

### 5. CSS Styling
- **File:** `static/css/components/autopilot.css`
- **Styles:** Progress modal, brief panel, run-now button
