# Plan 20-02: Autopilot API + Run Now

## Goal
Provide API endpoints for manual autopilot triggering and brief retrieval, with SSE progress streaming.

## Requirements Covered
- AUTO-03: User can trigger autopilot run manually via "Run Now"

## Tasks

### 1. Run Now Endpoint
- **File:** `src/api/autopilot.py`
- **Endpoint:** `POST /api/institutions/<id>/autopilot/run-now`
- **Action:** Trigger `execute_autopilot_run()` with `trigger_type=MANUAL`
- **Response:** `{"run_id": "apr_xxx", "status": "running"}`
- **Background:** Use threading to run async, return immediately

### 2. SSE Progress Stream
- **File:** `src/api/autopilot.py`
- **Endpoint:** `GET /api/institutions/<id>/autopilot/runs/<run_id>/progress`
- **Action:** Stream progress updates via SSE
- **Events:**
  - `progress`: `{"message": "...", "percent": 50}`
  - `complete`: `{"run_id": "...", "status": "completed", "score_after": 85}`
  - `error`: `{"error": "..."}`

### 3. Brief Retrieval Endpoints
- **File:** `src/api/autopilot.py`
- **Endpoints:**
  - `GET /api/institutions/<id>/briefs/latest` → most recent brief
  - `GET /api/institutions/<id>/briefs?days=30` → list of briefs

### 4. Brief Download
- **Endpoint:** `GET /api/institutions/<id>/briefs/<date>/download`
- **Response:** Markdown file download
