    **Harvesters package (src/harvesters/):**

    `__init__.py`: Export `HarvesterType`, `BaseHarvester`, `WebHarvester`, `PdfHarvester`, `ManualHarvester`, and a factory function `create_harvester(harvester_type: HarvesterType) -> BaseHarvester`.

    `base_harvester.py`:
    - `HarvesterType(str, Enum)` with values `WEB_SCRAPER = "web_scrape"`, `PDF_PARSER = "pdf_parse"`, `MANUAL_UPLOAD = "manual_upload"`
    - `class BaseHarvester(ABC)` with abstract method `fetch(self, source_config: Dict[str, Any]) -> Dict[str, Any]` returning `{"text": str, "metadata": dict}`

    `web_harvester.py`:
    - `class WebHarvester(BaseHarvester)` with `fetch(source_config)`:
      - `source_config` expects `{"url": str}`
      - Uses `requests.get(url, headers={"User-Agent": "AccreditAI Standards Harvester/1.0 (Educational Tool)"}, timeout=30)`
      - Parses with `BeautifulSoup(response.content, 'lxml')`
      - Extracts text from `soup.get_text(separator='\n', strip=True)` (flexible, not tied to specific selectors since ACCSC HTML structure may vary)
      - Also attempts to find main content via `soup.find('main')` or `soup.find('article')` or `soup.find('div', class_='content')` with fallback to full body text
      - Returns `{"text": extracted_text, "metadata": {"source_url": url, "title": soup.title.string if soup.title else "", "fetched_at": now_iso(), "encoding": response.encoding}}`
      - Rate limiting: `time.sleep(12)` before request per research recommendation
      - Raises `ValueError` on HTTP errors or empty content

    `pdf_harvester.py`:
    - `class PdfHarvester(BaseHarvester)` with `fetch(source_config)`:
      - `source_config` expects `{"file_path": str}` for local file OR `{"url": str}` for remote PDF
      - If `url` provided: downloads PDF to temp file with `requests.get(url, timeout=60)`, saves to `standards_library/downloads/` directory
      - Uses `pdfplumber.open(file_path)` to extract text page-by-page (same pattern as `document_parser.py` lines 155-207)
      - Concatenates pages with `"\n\n".join(pages_text)`
      - If text length < 100 chars, logs warning about possible image-based PDF
      - Returns `{"text": extracted_text, "metadata": {"file_path": str, "page_count": int, "extraction_method": "pdfplumber", "fetched_at": now_iso()}}`

    `manual_harvester.py`:
    - `class ManualHarvester(BaseHarvester)` with `fetch(source_config)`:
      - `source_config` expects `{"text": str, "version_date": str (optional), "notes": str (optional)}`
      - Simply returns `{"text": source_config["text"], "metadata": {"source_type": "manual_upload", "notes": source_config.get("notes", ""), "uploaded_at": now_iso()}}`

    **Versioning Service (src/services/standards_versioning_service.py):**
    - Import `hashlib`, `json`, `logging`, `os`, `sqlite3`, `uuid4`, `HtmlDiff` from difflib, `get_conn` from `src.db.connection`, `now_iso` from `src.core.models`
    - `normalize_text(text: str) -> str`: Strip whitespace, normalize `\r\n` to `\n`, collapse multiple blank lines
    - `compute_text_hash(text: str) -> str`: `hashlib.sha256(normalize_text(text).encode('utf-8')).hexdigest()`
    - `store_version(accreditor_code: str, text: str, source_type: str, source_url: str = None, metadata: dict = None, version_date: str = None) -> dict`:
      - Compute hash via `compute_text_hash(text)`
      - Generate ID with `str(uuid4())`
      - Save text to file at `standards_library/versions/{accreditor_code}/{version_id}.txt`
      - Create parent dirs with `os.makedirs(..., exist_ok=True)`
      - Insert row into `standards_versions` table
      - `version_date` defaults to `now_iso()` if not provided
      - Return version dict: `{"id": id, "accreditor_code": ..., "version_date": ..., "content_hash": ..., "source_type": ..., "is_new": bool, "changed": bool}`
    - `get_versions(accreditor_code: str) -> list`: SELECT all versions for accreditor ordered by `version_date DESC`
    - `get_latest_version(accreditor_code: str) -> dict | None`: SELECT latest version by `version_date DESC LIMIT 1`
    - `get_version_text(version_id: str) -> str | None`: Read text from file_path stored in version record
    - `detect_change(accreditor_code: str, new_text: str) -> dict`: Compare hash of new_text against latest version hash. Return `{"changed": bool, "is_new": bool, "previous_hash": str|None, "new_hash": str}`
    - `generate_diff(old_version_id: str, new_version_id: str) -> str`: Load text for both versions, use `HtmlDiff(wrapcolumn=80).make_table(old_lines, new_lines, fromdesc="Previous Version", todesc="Current Version", context=True, numlines=3)`. If old_version_id is None, return `'<div class="diff-info">First version - no previous version to compare</div>'`

    **API Blueprint (src/api/standards_harvester.py):**

    Follow the `init_*_bp(dependencies)` DI pattern from `src/api/standards.py`:
    ```python
    standards_harvester_bp = Blueprint('standards_harvester', __name__, url_prefix='/api/standards-harvester')
    _workspace_manager = None

    def init_standards_harvester_bp(workspace_manager):
        global _workspace_manager
        _workspace_manager = workspace_manager
        return standards_harvester_bp
    ```

    Endpoints:

    1. `POST /api/standards-harvester/fetch` — Trigger a harvest
       - Request JSON: `{"accreditor_code": str, "source_type": "web_scrape"|"pdf_parse"|"manual_upload", "url": str (optional), "text": str (optional), "version_date": str (optional)}`
       - Creates appropriate harvester via `create_harvester(HarvesterType(source_type))`
       - Calls `harvester.fetch(source_config)` to get text + metadata
       - Calls `detect_change(accreditor_code, text)` to check if new
       - Calls `store_version(accreditor_code, text, source_type, ...)` to persist
       - Returns 201: `{"version": {...}, "change_detected": bool}`
       - Returns 400 on invalid source_type or missing required fields
       - Returns 500 on fetch errors (scraping failures, PDF parse errors)

    2. `POST /api/standards-harvester/upload` — Upload a PDF file
       - Accepts multipart/form-data with `file` field and `accreditor_code` field
       - Saves file to `standards_library/uploads/{accreditor_code}/{filename}`
       - Uses PdfHarvester with `{"file_path": saved_path}`
       - Same version storage and change detection as /fetch
       - Returns 201: `{"version": {...}, "change_detected": bool}`

    3. `GET /api/standards-harvester/versions/<accreditor_code>` — List versions
       - Returns 200: `{"versions": [...]}`

    4. `GET /api/standards-harvester/versions/<accreditor_code>/latest` — Get latest version
       - Returns 200: `{"version": {...}}` or 404 if no versions

    5. `GET /api/standards-harvester/diff/<old_version_id>/<new_version_id>` — Get diff HTML
       - Returns 200: `{"diff_html": str, "old_version": {...}, "new_version": {...}}`

    6. `GET /api/standards-harvester/diff/<new_version_id>` — Get diff against previous version
       - Looks up version, finds previous version for same accreditor_code
       - Returns 200: `{"diff_html": str, "old_version": {...}|null, "new_version": {...}}`
