    **JavaScript (static/js/standards-harvester.js):**
    Class `HarvesterManager` with methods:

    ```javascript
    class HarvesterManager {
        constructor() {
            this.currentAccreditor = 'ACCSC';
            this.currentTab = 'web_scrape';
            this.versions = [];
        }

        init() // Called on DOMContentLoaded, binds events, loads versions
        switchTab(tab) // Show/hide source panels, update active tab
        async fetchWebScrape() // POST /api/standards-harvester/fetch with {accreditor_code, source_type: "web_scrape", url}
        async uploadPdf() // POST /api/standards-harvester/upload with FormData
        async submitManual() // POST /api/standards-harvester/fetch with {accreditor_code, source_type: "manual_upload", text}
        async loadVersions() // GET /api/standards-harvester/versions/{accreditor}
        renderVersionTable(versions) // Build table rows with date, source badge, hash, diff button
        async showDiff(versionId) // GET /api/standards-harvester/diff/{versionId}, inject diff_html into modal
        showDiffModal() // Show modal overlay
        hideDiffModal() // Hide modal overlay
        showLoading() // Show spinner
        hideLoading() // Hide spinner
        showResult(message, type) // Show result banner (success/error/info)
    }

    document.addEventListener('DOMContentLoaded', () => {
        window.harvesterManager = new HarvesterManager();
        window.harvesterManager.init();
    });
    ```

    Key behaviors:
    - On accreditor dropdown change: reload versions list
    - On tab click: switch visible form panel
    - On fetch/upload/submit: show loading, call API, show result banner, reload versions
    - On "View Diff" click: call diff endpoint, inject `diff_html` into modal container, show modal
    - Error handling: catch fetch errors, show error banner with message from API response
