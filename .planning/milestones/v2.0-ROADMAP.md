## Phase 38: Bulk Remediation Wizard

Transform batch operations into guided wizard:

- "Fix Everything" entry point from dashboard and work queue
- Scope selection (all docs, doc type, program)
- Priority queue display (critical → info)
- SSE progress with per-document status
- Batch approval UI with accept/reject per document

**Plans:**
- 38-01: Bulk remediation service + SSE streaming
- 38-02: Wizard UI with progress and batch approval

## Phase 39: Packet Studio Wizard

5-step visual wizard for submission packets:

1. Select submission type (self-study, response, teach-out)
2. Select standards/requirements
3. Map evidence to standards (with coverage validation)
4. Generate/review narratives
5. Preview + Export (with gating)

**Plans:**
- 39-01: Wizard backend (step state machine, preview rendering)
- 39-02: Wizard UI (5-step flow, drag-drop evidence mapping)

## Phase 40: Document Workbench IDE

Transform workbench into IDE-like experience:

- Three-panel layout (docs | content | findings)
- Inline finding markers in document view
- Click finding → highlight relevant text
- Fix button per finding → remediation
- Side-by-side original vs. remediated

**Plans:**
- 40-01: IDE backend (document rendering, finding overlay)
- 40-02: IDE UI (three-panel, inline markers, compare view)

## Success Criteria

- [ ] "Fix Everything" completes all remediations in priority order
- [ ] Packet wizard guides from selection to export
- [ ] Workbench shows findings inline with document text
